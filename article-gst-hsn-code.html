1. What is an HSN Code?

HSN is an internationally accepted system developed by the World Customs Organization (WCO) for classifying goods. Under GST, every goods invoice must mention the correct HSN code — this code determines the applicable GST rate. Services use SAC (Services Accounting Code) instead.

2. HSN Code Structure

Example: Cotton T-shirt — HSN 61091000

4. How to Find the Right HSN Code

  1. Go to gst.gov.in → Services → User Services → Search HSN Code
  2. Type your product name and browse the results
  3. Select the most specific classification at 6-digit or 8-digit level
  4. Alternatively use cbic-gst.gov.in → GST Rate Finder (shows rate directly)

Use the material/composition of the product, not just the common name. When in doubt between two codes, choose the more specific one.

7. Consequences of Wrong HSN Code

  • Wrong GST rate: Under-declared tax creates demand + interest + penalty
  • ITC rejection: Buyer's ITC claim may be rejected if your HSN is wrong
  • GSTR-1 mismatch: HSN summary mismatch triggers scrutiny notice
  • E-way bill errors: Wrong HSN in e-way bill can lead to goods detention in transit

8. HSN Summary in GSTR-1

GSTR-1 Table 12 requires HSN-wise summary of outward supplies. For turnover above ₹5 crore: 6-digit level. Common errors: total taxable value in HSN summary not matching supplies, same HSN with different rates not reported separately, exports without 8-digit HSN.
